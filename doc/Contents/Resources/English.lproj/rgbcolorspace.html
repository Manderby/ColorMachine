RGB Colorspace

The selection of an RGB colorspace defines the illumination, the response curve and the primaries R, G and B.

Several predefined RGB colorspaces can be selected. Additionally, Custom RGB colorspaces can be defined where the type of the response curve as well as the primaries for each of the primaries Red, Green and Blue, can be defined as Yxy values.

You can also set all three primaries with a menu command. Simply choose the desired color in any colorspace and choose the menu command or the appropriate keyboard shortcut.

When the response curve type Gamma is selected, an arbitrary gamma value can be chosen.

When selecting a predefined colorspace, the illumination is automatically set to the correct choice. When the illumiation is changed, the RGB colorspace is automatically changed to Custom

The Y component of the Yxy values of the primaries are computed automatically such that summing up all primaries results in the colorimetric whitepoint.
