Lab Colorspace

The Lab Colorspace can be set to several predefined colorspaces as well as two custom spaces:

  • CIELAB: The default Lab colorspace used nowadays.
  • Custom L: The same as CIELAB but with an arbitrary L response. The a and b components are computed accordingly.
  • Hunter Lab Approximated: The Hunter Lab space where the constants K and ke are computed according to the whitepoint.
  • Hunter Lab Original: The Hunter Lab space with fixed constants K and ke.
  • Adams chromatic valence: An arbitrary Adams chromatic valence space with custom constants K and ke.

When the Custom L space is selected, the response curve can arbitrarily be chosen. When the response curve type Gamma is selected, an arbitrary gamma value can be chosen.

When the Adams chromatic valence space is selected, the constants K and ke can be changed arbitrarily.
